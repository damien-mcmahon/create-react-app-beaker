Create React App for Beaker

If you want to create sites in Beaker and want to be using React then C.R.A.B has got you sorted.

Fork this site and follow the instructions below to get started:

  1. Fork this site
  2. Give it a name etc
  3. Go to the Site using a terminal
  4. Run the following commands
    yarn
    This will install all the dependencies needed.
    yarn build
    This will run the yarn build command
    yarn dev
    This will watch the build folder and rebuild when changes are detected
    Change Folder in the library
    Set your build folder as the root
    Do your thing!
    Make changes to the code under 'src' and you'll see it change live in Beaker

Want to make this better?

This is a simple starter for CRA and Beaker. I want to make it more automated but for now there's those steps up there. If you want to help then feel free to fork the repo on Github and make things better!
